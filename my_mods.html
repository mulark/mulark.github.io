My Mods

Source code for all mods is available at https://github.com/mulark/factorio_benchmarking_mods

Chunk Paver Mod

This mod converts land tiles on newly generated chunks into refined concrete. Lakes and oceans are not covered by refined concrete.

Rock and Fish Remover Mod

This mod removes all rocks and fish on newly generated chunks. This reduces the save file size by a bit.
