My Mods

Source code for all mods is available at https://github.com/mulark/factorio_benchmarking_mods

Creative World Plus

Converts the game world into refined concrete. Compared to the original Creative World this mod is far more UPS friendly.

Rock and Fish Remover Mod

This mod removes all rocks and fish on newly generated chunks. This reduces the save file size by a bit. Trees should be disabled in the map generator settings.

This mod is DEPRECATED, since it makes more sense to use creative world plus

Region Cloner Mod

This mod copies entites in one region into another region(s). It can handle trains, cars, belts, items on a belt, fuel levels, fluid, and more. Documentation.
